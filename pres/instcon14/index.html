Techrangers Team

  • Founded in 1996 to help faculty create online courses
  • 11 students working part-time
  • Primary task is maintaining online course content
  • Accessibility of content is a major concern
  • We also create web applications

The Challenge

Before Each Semester

  • Identify courses requiring accommodation.
  • Perform accessibility evaluations.
  • If needed, repair course content.

Why not fix them all?

  • Not enough resources
  • Moving target
  • We should treat the illness, not the symptom

What's the illness?

  • Canvas makes it easy to create and edit content, but...
  • Faculty lack training in universal design.
  • Accessibility issues will continue to appear.

Our Solution

Our Inspiration

  • The WAVE Tool (wave.webaim.org)
  • W3C HTML Validator (validator.w3.org)

Project Goals

  • Self-service (You do it!)
  • Will "scan" online course content for common accessibility issues
  • Will provide further explanation of the issues discovered, with steps the instructor can take to fix the problem themselves.
