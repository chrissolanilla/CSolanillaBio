How Do I Apply to be a Techranger?

If you would like to join the Techrangers' team, please do the following:

  • Update your online portfolio to showcase any relevant projects
  • Create a resume (HTML strongly preferred)

Click here to apply

Applications will be accepted until all positions are filled. The top applicants will be invited to a group interview. Hired Techrangers will need to be available to begin work immediately for a one year minimum. All Techrangers are required to work between 15 to 20 hours a week, and must be currently enrolled as students at UCF.

Questions? Email us.

What skills do I need to be a Techranger?

All Techrangers should possess the following qualities and skills:

Core Skills

  • Intermediate to advanced XHTML knowledge
  • Intermediate CSS knowledge
  • Basic programming knowledge
  • Basic JavaScript knowledge
  • Basic web graphics experience
  • Windows OS

Professional Skills

  • Team player
  • Verbal and Written Communication
  • Eagerness to Learn
  • Initiative
  • Presentational
  • Organizational
  • Leadership
  • Time Management
  • Problem Solving
  • Sense of Humor
  • Creativity
  • Patience

Technical Skills

Knowledge of one or more of the following is a plus:

  • Javascript/jQuery
  • MySQL
  • PHP
  • Python
  • Dreamweaver
  • Graphic Design
  • User Centered Design
  • Accessibility
  • Browsers
    • Opera
    • Safari
    • Internet Explorer
    • Firefox
  • Operating Systems
    • Linux
    • Mac OS
