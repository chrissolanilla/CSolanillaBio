Showcase

Materia

This tool allows direct access between Materia and Webcourses@UCF. Materia is a platform for creating customizable games and study tools for online courses. Examples include Hangman, Word Search, Matching, and Adventure.


Visit the Materia site for more information.

Obojobo

This integration allows direct access from the online course to Obojobo. With Obojobo, you can create focused learning objects that can be presented to students in your online course. The learning objectives created in Obojobo can be linked to the gradebook in Webcourses@UCF for automatic grading.


Visit the Obojobo site for more information.

UDOIT

This tool enables faculty to identify accessibility ssues on Webcourses@UCF. It will scan a course, generate a report, and provide resources on how to address common accessibility issues. In many cases, it will enable you to easily fix the issue.


Visit the UDOIT site for more information.
